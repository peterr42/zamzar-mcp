# mcp-poc
Simple POC for Zamzar MCP server (for integration with Claude Code), programmed in PHP.


## Setup Guide:
- download or clone this repo
- open the Claude Desktop App
- navigate to Settings -> Extensions -> Advanced Settings -> Install Extension
- locate the extension.mcpb file included in this repo
- select it and click the preview button
- install the zamzar-mcp extension
- create a new chat or open an existing chat with Claude
- ask it for the MCP tools offered by Zamzar

This will list the available tools (currently just converting files and viewing Zamzar account information) for use. Ask it to "run the Zamzar convert function" for a test.
